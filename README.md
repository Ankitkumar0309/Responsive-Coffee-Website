☕ Coffee Shop Website

A modern and responsive coffee shop website built using HTML, CSS, and JavaScript.

📌 Overview

This project is a coffee shop landing page designed to provide an engaging and visually appealing experience for visitors. It showcases coffee products, services, and essential business information with a clean and responsive design.

✨ Features

- Modern and attractive UI
- Fully responsive design
- Smooth navigation
- Coffee menu showcase
- About Us section
- Contact section
- Mobile-friendly layout
- Clean and organized code

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
